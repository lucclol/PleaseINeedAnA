# Give Evan an A

Voice-controlled rhythm game. Say "LEFT", "DOWN", "UP", "RIGHT" when arrows hit the zone — or use keyboard arrows / D F J K.

## Deploy

Push to GitHub and connect to Vercel. The `public/` folder is the output directory.

## Local Dev

```bash
cd public
python -m http.server 8000
# open http://localhost:8000
```

## Files

- `public/index.html` — The game
- `public/music.mp3` — Background track (120 BPM, NO COPYRIGHT)
